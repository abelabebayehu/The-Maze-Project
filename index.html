About The Maze Project

This project was created as part of my ALX curriculum, with the goal of building a maze generator that is both fun and challenging. The project was inspired by my interest in algorithm design and gaming. It showcases my skills in C programming, problem-solving, and using SDL for graphical rendering.

As a civil engineer transitioning into software development, this project is a major milestone in my learning journey.
