Key Features

3D Raycasting

Utilizes raycasting to render a 3D environment for navigating mazes, creating an immersive experience.

Maze Exploration

Explore complex, randomly generated mazes. Every maze is unique, providing a different challenge each time you play.

First-Person View

Experience the maze from a first-person perspective, enhancing the sense of immersion and engagement.

About The Maze Project

The Maze Project was created as part of my ALX curriculum, where I challenged myself to create a 3D maze game using raycasting. The inspiration for this project stemmed from my love for algorithm design and gaming. This is a portfolio project showcasing my skills in C programming, game development, and graphical rendering using SDL.
